Conceptos basicos

Algoritmo Secuencia de instrucciones no ambiguas para resolver un problema dentro de un marco de tiempo finito
Entrada El dato (o datos) que recibe el algoritmo. Puede ser el tamaño de un arreglo, la cantidad de palabras, etc. Es representado por n
Complejidad temporal Tiempo maximo que requiere un algoritmo de inicio a fin. Se puede medir en segundos o milisegundos y es representado como O(n)

Que es Fuerza Bruta?

La fuerza bruta (Brute Force) es el metodo más sencillo de implementar para diseñar algoritmos para solucionar problemas. Se baza en revisar cada posible opcion dentro de un problema hasta encontrar con la solucion.

Aunque es el metodo más facil de implementar, tambien es el menos efectivo, debido a que debe revisar elemento por elemento

Resolver Problemas

Busqueda de patrones

Conocido como Busqueda Bruta (Brute Search), este algoritmo recibe una cadena de caracteres, conocida como el Haystack, y una palabra, conocida como el Needle. El objetivo es encontrar la palabra dada dentro de la cadena.

Como ya se mencionó, por medio de Fuerza Bruta se debe analizar posicion por posicion no solo el Haystack, pero tambien el Needle, lo que causa que este algoritmo tenga una complejidad de O(n+m), donde n es el tamaño del Haystack y m el tamaño del Needle
